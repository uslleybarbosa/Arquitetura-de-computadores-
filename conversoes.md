## Converta estes números de decimal para binário

<details>
  <summary>🔢 Conversão decimal → binário (100)</summary>

```asm
Resolução:  
100 / 2 = 50 → resto = 0  
50 / 2 = 25 → resto = 0  
25 / 2 = 12 → resto = 1  
12 / 2 = 6  → resto = 0  
6 / 2 = 3   → resto = 0  
3 / 2 = 1   → resto = 1  
1 / 2 = 0   → resto = 1
```
#### Resposta:

🔹 O número 100 em decimal é igual a 1100100 em binário.

## Converta estes números de octal para hexadecimal

<details>
  <summary>🔢 Conversão octal → hexadecimal (174)</summary>

```asm
Resolução:  
174₈ = 1 * 8² + 7 * 8¹ + 4 * 8⁰  
= 64 + 56 + 4  
= 124₁₀  

124 / 16 = 7 → resto = 12 = C
```
#### Resposta:

🔹 O número 174 em octal é igual a 7C em hexadecimal.
</details>

<details>
  <summary>🔢 Conversão octal → hexadecimal (4321)</summary>

= 2257₁₀  

2257 / 16 = 141 → resto = 1

## Converta estes números de binário para hexadecimal

<details>
  <summary>🔢 Conversão binário → hexadecimal (101010)</summary>

```asm
Resolução:  
101010₂ = 32 + 8 + 1  
= 42₁₀  

42 ÷ 16 = 2 → resto = 10 → A  
2 ÷ 16  = 0 → resto = 2
```
#### Resposta:

🔹O número 101010 em binário é igual a 2A em hexadecimal.

</details>
