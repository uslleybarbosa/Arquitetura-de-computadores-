## Componentes básicos do computador

estudaremos os principais componentes de um computador, como a CPU, memória primária (RAM e cache), memória secundária e dispositivos de entrada e saída (I/O).


### 🧠 CPU (Unidade Central de Processamento)

A CPU, também chamada de processador, é o “coração” do computador. Ela é responsável por executar instruções e processar dados.

Dentro da CPU existem:

* **Registradores**: pequenas áreas de memória muito rápidas usadas para armazenar dados temporários durante o processamento.
* **Clock**: controla a velocidade das operações.


### 💾 Memória Primária

A memória primária é composta pela memória RAM, onde os dados são armazenados temporariamente enquanto o computador está em funcionamento.

* A RAM se comunica diretamente com a CPU.
* Tipos de RAM:
  
  - **SDRAM**: mais antiga, realiza uma operação por ciclo de clock.
  - **DDR**: mais moderna e rápida, realiza mais de uma operação por ciclo.

### ⚡ Memória Cache

A memória cache é mais rápida que a RAM e é usada para armazenar dados frequentemente acessados pela CPU.

Ela é dividida em níveis:

* **L1** (mais rápida)
* **L2**
* **L3** (maior, porém mais lenta que L1)

### 🗄️ Memória Secundária

A memória secundária é responsável pelo armazenamento permanente dos dados, mesmo sem energia.

Exemplos:

* **HD (Disco Rígido)**
* **SSD (Unidade de Estado Sólido)**

### ⌨️ Dispositivos de Entrada e Saída (I/O)

Os dispositivos de entrada e saída permitem a interação entre o usuário e o computador.

Exemplos:

* Teclado
* Mouse
* Impressora

Em alguns casos, utiliza-se o **DMA (Acesso Direto à Memória)**, que permite a transferência de dados diretamente entre dispositivos e a memória, sem sobrecarregar a CPU.
